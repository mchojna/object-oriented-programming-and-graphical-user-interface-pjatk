Zdefiniować klasę Osoba zawierającą prywatne pola imie typu String, wiek typu int wraz z konstruktorem, getterami i metodą toString().  Klasa Osoba powinna implementować interfejs funkcyjny Comparable<Osoba> czyniąc obiekty-osoby porównywalnymi według następujących kryteriów uporządkowania: wśród osób porównujemy najpierw alfabetycznie imiona, potem wiek (tj. kiedy imiona są równe).

Następnie zdefiniować klasę KomparatorOsob implementującą interfejs funkcyjny java.util.Comparator<Osoba>. Klasa KomparatorOsob powinna zawierać konstruktor z argumentem typu wylicznikowego (enum) Kryterium określającym kryterium, według którego obiekty-osoby będą uporządkowane.

Stworzyć wszystkie klasy w taki sposób, aby następujący program:

import java.util.*;

public class Test {
   
    public static void main(String[] args) {
            
            // tworzenie listy osób
        List<Osoba> lista = Arrays.asList(
                                new Osoba("Anna", 23),
                                new Osoba("Maria", 22),
                                new Osoba("Anna", 20),
                                new Osoba("Wojciech", 21)
                            );
            // sortowanie według podanego komparatora (po imię)
        Collections.sort(lista, new KomparatorOsob(Kryterium.imie));
        System.out.println(lista);
        
            // sortowanie według podanego komparatora (po wieku)
        Collections.sort(lista, new KomparatorOsob(Kryterium.wiek));
        
            // lub zamiast komparatora "po wieku" napisać wyrażenie lambda w miejscu /* ... */
        Collections.sort(lista, /* ... */);
        System.out.println(lista);
        
            // sortowanie według porządku naturalnego określonego w klasie Osoba
        Collections.sort(lista);
        System.out.println(lista);
    }

}

Zadanie 12

Każdy student posiada nazwisko, imię, rok urodzenia oraz unikalny numer. Każda grupa posiada swoją nazwę oraz maksymalną liczbę studentów. Początkowo student nie jest przypisany do żadnej grupy. Studenta można przypisać do podanej (tylko jednej!) grupy. Studenta można usunąć z grupy, jeżeli wcześniej był przypisany do niej.

Wykorzystując klasy kolekcyjne (listy) napisać program przypisując studentów do grup oraz zmieniając ich grupy. Stworzyć klasy w taki sposób, aby następujący program


public class StudentGroup {

    public static void main(String[] args) {

        Student s1 = new Student("Kowalski", "Jan", 2000);
        Student s2 = new Student("Nowacka", "Janina", 2005);
        Student s3 = new Student("Omega", "Alfa", 2003);

        Group gr1 = new Group("Java", 15);
        Group gr2 = new Group("Smalltalk", 2);

        try {
            gr1.assign(s1);
            gr1.assign(s3);
            gr1.assign(s2);

            System.out.println(gr1);
            for (Student s : gr1.getStudents()) System.out.println(s);

            gr2.assign(s1);
        } catch (Exception e) {
            System.out.println(e.getMessage());
        }

        try {
            for (Student s : gr1.getStudents()) gr1.remove(s);
            gr2.assign(s1);
            gr2.assign(s2);

            System.out.println(gr2);
            for (Student s : gr2.getStudents()) System.out.println(s);

            gr2.assign(s3);
        } catch (Exception exc) {
            System.out.println(exc.getMessage());
        }

        try {
            gr2.remove(s3);
        } catch (Exception exc) {
            System.out.println(exc.getMessage());
        }
    }
}
